# DevOps Capstone Project

![CI Build](https://github.com/Monesh391/devops-capstone-project/actions/workflows/ci-build.yaml/badge.svg)

## Overview

This repository contains the DevOps Capstone Project developed as part of the Coursera DevOps Professional Certificate.

## Features

- Agile project planning using GitHub Projects
- RESTful Accounts microservice
- Unit testing with Nose
- Code quality checks using Flake8 and Pylint
- Continuous Integration using GitHub Actions
- Security headers and CORS policies
- Docker containerization
- Kubernetes deployment
- Tekton CI/CD pipeline

## Technology Stack

- Python
- Flask
- Docker
- Kubernetes
- GitHub Actions
- Tekton
- GitHub
